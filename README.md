# Nord-Norge — KOSTRA, ROBEK & demografi

Interaktivt, frittstående dashbord for de 80 kommunene i Nord-Norge (Nordland, Troms, Finnmark) etter kommune- og regionreformen 2024.

**Innhold:** demografisk framskriving (kohort-komponentmodell, SSB-basert), kommuneøkonomi (KOSTRA), og ROBEK-status med kalibrert risikoproxy på de økonomiske fundamentene.

- `index.html` — hele applikasjonen i én fil (HTML/JS, Chart.js). Ingen byggesteg.
- Datakilde: Statistisk sentralbyrå (SSB), PxWebApi v2.

Åpne `index.html` i nettleser, eller se den publiserte versjonen (Vercel).

> Risikoproxyen er et gjennomsiktig tidligvarsel på KOSTRA-tall, **ikke** de juridiske ROBEK-vilkårene (kommuneloven §28-1). Registerstatus er et øyeblikksbilde.
